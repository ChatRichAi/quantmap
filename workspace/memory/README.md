# QuantClaw Memory Layout

## 目录说明

- `daily/`: 每日交易与复盘日志
- `trades/`: 结构化交易记录（JSON）
- `strategies/`: 策略表现与回顾
- `rebalance/`: 再平衡检查与执行记录
- `research/`: 研究与实验结论
- `state/`: 监控与执行状态文件
- `templates/`: 报告模板

## 标签规范（Nowledge Mem）

- `trade-record`
- `strategy-performance`
- `market-observation`
- `risk-event`
- `rebalance-action`
- `research-finding`
