# technical-analysis

## 整合来源

- `skills/stock-sniper`
- `skills/chaoduan-strategy`

## 分析维度

- 趋势: MA/EMA 结构
- 动量: RSI/MACD
- 波动: ATR/区间振幅
- 结构: 支撑阻力/突破回踩
- 共振: 15分钟、1小时、4小时、1天K线数据趋势一致性

## 输出模板

1. 市场状态
2. 关键信号
3. 失效条件
4. 风险等级
5. 跟踪建议

## 注意事项

- 不输出无条件的“立即买卖”建议
- 强制给出失效价位或失效事件
