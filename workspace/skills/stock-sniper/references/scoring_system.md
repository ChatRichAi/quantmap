# 加权评分体系

## 综合评分计算方式

总分 = 六要素平均分 + 数据质量加成 - 风险扣分

### 六要素权重分配

| 要素 | 权重 | 说明 |
|------|------|------|
| 指数环境 | 15% | 大盘环境决定仓位基础 |
| 主线题材 | 25% | 题材强度是超短核心 |
| 情绪周期 | 20% | 择时的关键 |
| 个股定位 | 20% | 个股在板块中的地位 |
| 风偏识别 | 10% | 与市场风格匹配度 |
| 联动关系 | 10% | 板块内协同效应 |

### 数据质量加成

| 数据完整度 | 加分 |
|------------|------|
| akshare + TV + 新闻 + 舆情 | +0.5 |
| 缺少一项数据源 | +0.3 |
| 缺少两项以上 | +0 |

### 风险扣分项

| 风险类型 | 扣分 |
|----------|------|
| 指数暴跌（-2%以上） | -1.0 |
| 高位放量滞涨 | -0.5 |
| 龙虎榜无游资 | -0.3 |
| 负面新闻 | -0.5~-2.0 |

## 操作建议映射

| 总分范围 | 操作建议 | 仓位建议 |
|----------|----------|----------|
| 9-10 | 强力买入 | 重仓 |
| 8-8.9 | 积极买入 | 半仓以上 |
| 7-7.9 | 关注买入 | 半仓 |
| 6-6.9 | 轻仓试错 | 轻仓 |
| 5-5.9 | 观望 | 不参与或极小仓位 |
| <5 | 放弃 | 不参与 |

## 各要素详细评分标准

### 1. 指数环境 (0-10分)

| 情况 | 得分 | 说明 |
|------|------|------|
| 指数主升浪+放量 | 9-10 | 满仓做多环境 |
| 指数震荡向上 | 7-8 | 可参与，控制仓位 |
| 指数横盘震荡 | 5-6 | 精选个股，快进快出 |
| 指数下跌通道 | 2-4 | 空仓或极小仓位 |
| 指数暴跌 | 0-1 | 坚决空仓 |

### 2. 主线题材 (0-10分)

| 情况 | 得分 | 说明 |
|------|------|------|
| 国家级大题材+批量涨停 | 9-10 | 必须参与 |
| 行业级利好+龙头明确 | 7-8 | 重点参与 |
| 事件驱动+有持续性 | 5-6 | 可参与 |
| 纯概念炒作 | 3-4 | 谨慎参与 |
| 无题材支撑 | 0-2 | 放弃 |

### 3. 情绪周期 (0-10分)

| 阶段 | 得分 | 策略 |
|------|------|------|
| 冰点试错 | 8-9 | 轻仓试错了 |
| 回暖期 | 9-10 | 积极参与 |
| 高潮期 | 6-7 | 持股或减仓 |
| 退潮期 | 2-4 | 空仓等待 |
| 恐慌期 | 3-5 | 等冰点信号 |

### 4. 个股定位 (0-10分)

| 定位 | 得分 | 说明 |
|------|------|------|
| 板块总龙头 | 9-10 | 首选标的 |
| 板块龙二/龙三 | 7-8 | 次选标的 |
| 补涨龙 | 6-7 | 套利标的 |
| 跟风票 | 3-5 | 不参与 |
| 杂毛 | 0-2 | 坚决回避 |

### 5. 风偏识别 (0-10分)

| 市场风格 | 得分条件 | 说明 |
|----------|----------|------|
| 连板风偏 | 连板票+3个以上 | 做连板接力 |
| 趋势风偏 | 趋势票多+换手健康 | 做趋势低吸 |
| 反包风偏 | 反包票多+成功率高 | 做断板反包 |
| 20cm风偏 | 创业板涨停多 | 做弹性套利 |

### 6. 联动关系 (0-10分)

| 情况 | 得分 | 说明 |
|------|------|------|
| 龙头带板块，板块助龙头 | 9-10 | 最佳联动 |
| 龙头独强，板块跟涨 | 6-8 | 尚可 |
| 龙头弱，板块强 | 5-6 | 龙头可能切换 |
| 各自为战，无联动 | 2-4 | 题材不成立 |
