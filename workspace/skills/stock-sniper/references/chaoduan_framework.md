# 超短策略详细框架

## 一、盯盘流程详解

### 盘前准备 (8:45-9:15)

**核心任务**: 建立交易预案

1. **回顾昨日**: 昨日持仓、计划执行情况、市场走势回顾
2. **消息收集**: 
   - 隔夜美股走势
   - 重要政策消息
   - 个股公告/龙虎榜
   - 期货/外汇联动
3. **预案制定**: 根据消息和昨日走势，制定2-3套交易预案

### 隔夜单观察 (9:15-9:20)

**关键观察点**:
- 封单量大小（是否超预期）
- 隔夜单变化趋势（增加/减少）
- 是否有新题材被顶一字

**操作**: 将超预期个股加入自选池

### 竞价筛选 (9:20-9:25)

**筛选标准**:
1. **竞价成交额** > 5000万（活跃度高）
2. **竞价涨幅** 3%-8%（有资金关注但不过度）
3. **竞价量** > 昨日10%（资金抢筹迹象）
4. **封单异常** 一字板封单变化

### 开盘验证 (9:30-10:00)

**黄金30分钟原则**: 10点前强势票该封板的都封板了

**观察重点**:
- 开的强，走的强 = 真的强（可能追）
- 开的强，走弱 = 出货（回避）
- 开的弱，走强 = 分歧转一致（最佳买点）
- 开的弱，走弱 = 真的弱（放弃）

## 二、六要素深度解析

### 1. 指数分析框架

**三大指数关注**:
- 上证指数（权重、情绪）
- 创业板指（题材、弹性）
- 科创板指（科技、成长）

**指数K线语言**:
- **首阳**: 止跌信号，可能开启反弹
- **突破**: 趋势确认，积极做多
- **十字星**: 变盘信号，等待方向
- **长上影**: 抛压大，谨慎
- **长下影**: 支撑强，关注

**量能配合**:
- 上涨+放量 = 健康
- 上涨+缩量 = 背离
- 下跌+放量 = 恐慌
- 下跌+缩量 = 企稳

### 2. 题材分析框架

**题材生命周期**:
```
启动 → 发酵 → 高潮 → 分化 → 退潮 → 结束
```

**不同阶段策略**:
- **启动期**: 试错龙头，轻仓参与
- **发酵期**: 确认龙头，加仓参与
- **高潮期**: 持股或减仓，不追高
- **分化期**: 去弱留强，聚焦核心
- **退潮期**: 空仓等待，回避接力

**题材强度判断**:
| 指标 | 强 | 中 | 弱 |
|------|----|----|----|
| 涨停家数 | >20 | 10-20 | <10 |
| 连板高度 | >5板 | 3-5板 | <3板 |
| 龙头封单 | >10亿 | 5-10亿 | <5亿 |
| 板块涨幅 | >5% | 3-5% | <3% |

### 3. 情绪周期判断

**情绪周期四阶段**:

#### 冰点期
**特征**:
- 涨停家数 < 30
- 连板高度 < 3
- 炸板率 > 50%
- 跌停家数 > 10

**策略**: 空仓等待，或极小仓位试错

#### 回暖期
**特征**:
- 出现地天板
- 跌停减少
- 新题材启动
- 资金试探性入场

**策略**: 积极参与新题材

#### 高潮期
**特征**:
- 涨停家数 > 100
- 连板高度 > 7
- 遍地涨停
- 容错率高

**策略**: 持股待涨，不追高后排

#### 退潮期
**特征**:
- 高标断板
- 炸板率上升
- 天地板出现
- 跟风股大跌

**策略**: 减仓或空仓

### 4. 个股定位方法

**梯队划分**:
```
总龙头 → 板块龙 → 跟风股 → 补涨股 → 杂毛
```

**龙头识别五要素**:
1. **高度**: 谁高谁是龙
2. **强度**: 竞价封单、分时走势
3. **带动性**: 带动板块、指数
4. **抗跌性**: 分歧时能扛住
5. **辨识度**: 市场关注度最高

**买入优先级**:
1. 总龙头（首选）
2. 板块龙二（次选）
3. 换手龙（稳健选择）
4. 20cm弹性（套利）
5. 跟风股（放弃）

### 5. 风偏识别

**市场风格演变规律**:
- 4-5月: 连板风偏（小票接力）
- 6-7月: 趋势风偏（中军抱团）
- 8月: 机构风偏（大票趋势）

**风格判断指标**:
- 连板股数量
- 20cm涨停数量
- 大票/小票涨跌幅对比
- 龙虎榜游资/机构占比

**匹配策略**:
| 风偏 | 操作策略 |
|------|----------|
| 连板风 | 做连板接力，打板买入 |
| 趋势风 | 做低吸高抛，不破均线持有 |
| 反包风 | 做断板反包，核按钮低吸 |
| 20cm风 | 做弹性套利，快进快出 |

### 6. 联动关系观察

**三种联动模式**:

#### 龙头带小弟
- 龙头封板 → 小弟跟风上涨
- 龙头炸板 → 小弟跳水
- 策略: 做龙头

#### 小弟推龙头
- 小弟批量涨停 → 龙头被动上板
- 龙头炸板 → 小弟继续冲
- 策略: 谨慎参与

#### 平行联动
- 多只核心股同步上涨
- 相互助攻，共同走强
- 策略: 选最强的

## 三、节点交易法

### 三种关键节点

#### 1. 连板低切节点
**条件**:
- 高标断板或分歧
- 低位出现超预期
- 龙虎榜游资切换

**买点**: 低位超预期个股的首板或二板

#### 2. 总龙切换节点
**条件**:
- 旧总龙连续下跌
- 某股逆势走强
- 新题材启动

**买点**: 新龙头的二板确认

#### 3. 共振指数节点 ⭐最重要
**条件**:
- 指数关键K线（首阳、突破）
- 题材批量涨停
- 成交量放大

**买点**: 与指数共振的题材龙头

## 四、风险控制

### 止损原则
1. **单笔亏损** < 总资金3%
2. **日回撤** < 总资金5%
3. **周回撤** < 总资金10%

### 常见陷阱回避
- ❌ 高位放量滞涨
- ❌ 退潮期接力中位股
- ❌ 板块已高潮后追高
- ❌ 无题材支撑的纯技术反弹
- ❌ 指数暴跌时逆势重仓

### 仓位管理
- 指数主升: 重仓（80%+）
- 指数震荡: 中等仓位（50%）
- 指数下跌: 轻仓（20%）或空仓
- 情绪冰点: 空仓或极小仓位试错
