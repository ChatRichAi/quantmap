# 股吧情绪分析智能体 v3.0 - 三平台整合

整合**东方财富、雪球、淘股吧**三大平台的情绪数据，全面把握不同投资者群体的情绪动向。

## 数据来源

| 平台 | 数据类型 | 状态 | 用户特点 | 权重 |
|------|---------|------|---------|------|
| **东方财富** | 综合评分/关注度/参与度 | ✅ 真实数据 | 散户为主 | 40% |
| **雪球** | 专业文章/讨论 | 🎮 演示模式 | 专业投资者 | 35% |
| **淘股吧** | 帖子/涨停分析 | 🎮 演示模式 | 超短交易者 | 25% |

## 功能特点

- **三平台整合**: 散户 + 专业投资者 + 超短资金三维视角
- **权重计算**: 根据平台特点设置不同权重
- **综合情绪**: 加权计算综合情绪指数
- **差异化分析**: 不同平台的情绪特征对比

## 权重设计原理

### 为什么这样设置权重？

| 平台 | 权重 | 理由 |
|------|------|------|
| **东方财富 40%** | 最高 | 散户基数最大，反映大众情绪 |
| **雪球 35%** | 次高 | 专业投资者多，基本面分析深入 |
| **淘股吧 25%** | 较低 | 超短资金情绪，波动大但参考性强 |

### 权重调整

编辑 `scripts/multi_platform_analyzer.py` 中的 `weights` 字典：

```python
weights = {
    'eastmoney': 0.40,  # 调整这里
    'xueqiu': 0.35,
    'taoguba': 0.25,
}
```

## 情绪等级

| 综合指数 | 等级 | 交易参考 |
|---------|------|---------|
| > +40 | 🔥 极度乐观 | 三平台一致看多，警惕过热 |
| +20 ~ +40 | 📈 乐观 | 整体情绪向好，可持有 |
| +5 ~ +20 | ↗️ 偏多 | 情绪偏积极，谨慎参与 |
| -5 ~ +5 | 😐 中性 | 分歧较大，观望为主 |
| -20 ~ -5 | ↘️ 偏空 | 情绪偏谨慎，控制仓位 |
| -40 ~ -20 | 📉 悲观 | 整体偏空，注意风险 |
| < -40 | ❄️ 极度悲观 | 三平台一致看空，等反弹 |

## 平台特色解读

### 东方财富 - "散户风向标"

- **高关注指数** = 散户关注度高，人气旺
- **高参与度** = 交易活跃，流动性好
- **适用场景**: 判断股票热度、散户情绪

### 雪球 - "基本面雷达"

- **专业文章** = 深度基本面分析
- **大V观点** = 机构投资者动向
- **适用场景**: 判断股票质地、长期价值

### 淘股吧 - "超短温度计"

- **涨停分析** = 短线资金态度
- **连板预期** = 超短接力意愿
- **适用场景**: 判断涨停持续性、接力风险

## 实战技巧

### 三平台共振

当三个平台情绪一致时，信号最强：

| 情况 | 东财 | 雪球 | 淘股吧 | 信号强度 |
|------|------|------|--------|---------|
| 三多共振 | 看多 | 看多 | 看多 | ⭐⭐⭐⭐⭐ |
| 两多一空 | 看多 | 看多 | 看空 | ⭐⭐⭐ |
| 多空分歧 | 看多 | 看空 | 中性 | ⭐⭐ |
| 三空共振 | 看空 | 看空 | 看空 | ⭐⭐⭐⭐⭐ (反向) |

### 背离识别

- **东财热 + 雪球冷**: 散户热炒但机构不认可，警惕
- **东财冷 + 雪球热**: 机构看好但散户 unaware，机会
- **淘股吧极端**: 超短情绪过热/过冷，注意反转

### 时间窗口

- **盘前**: 看雪球隔夜分析 + 淘股吧涨停预期
- **盘中**: 看淘股吧实时讨论 + 东财热度变化
- **盘后**: 三平台综合分析，制定次日策略
