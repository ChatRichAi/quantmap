# 股吧情绪分析 - 实战案例参考

## 案例一：涨停板情绪监测

### 场景
某股昨日首板，今日计划二板接力。需要判断开盘情绪是否支持。

### 操作
```bash
# 盘前检查（9:00）
python scripts/analyze_sentiment.py --code 600875 --pages 3
```

### 结果解读
- **情绪指数 +35**：开盘预期乐观
- **看多帖占比 60%**：散户看好
- **关键词："涨停"、"龙头"、"冲"**

### 决策
结合竞价情况，如果竞价高开 + 情绪乐观，可尝试板上确认。

## 案例二：断板反包情绪判断

### 场景
昨日涨停股今日断板，判断是否有反包可能。

### 操作
```bash
# 盘中监控（10:00）
python scripts/analyze_sentiment.py --code 600875 --watch --interval 180
```

### 情绪变化
| 时间 | 情绪指数 | 判断 |
|------|---------|------|
| 9:30 | +20 | 乐观 |
| 10:00 | -10 | 转空 |
| 10:30 | -25 | 恐慌 |

### 决策
情绪从乐观快速转空，反包概率降低，考虑离场。

## 案例三：全市场热点捕捉

### 场景
寻找当日市场最热门的板块和个股。

### 操作
```bash
python scripts/analyze_sentiment.py --market --pages 5
```

### 分析维度
1. **出现频率**：哪些股票被提及最多
2. **情绪倾向**：整体看多还是看空
3. **关键词聚类**：发现热点概念

## 情绪与走势对照表

| 走势 | 情绪 | 操作建议 |
|------|------|---------|
| 高开高走 | 极度乐观(>+40) | 谨慎追，可能高潮 |
| 高开低走 | 极度乐观(>+40) | 警惕出货 |
| 低开高走 | 极度悲观(<-40) | 关注反弹机会 |
| 低开低走 | 极度悲观(<-40) | 恐慌盘，等企稳 |
| 震荡整理 | 中性(-5~+5) | 观望，等方向 |

## 常见误区

### 1. 情绪滞后
股吧情绪往往滞后于股价，不要只看情绪做决策。

### 2. 样本偏差
股吧以散户为主，大资金动向无法体现。

### 3. 极端情绪陷阱
极度乐观/悲观时往往是反向信号，不要追涨杀跌。

## 进阶用法

### 多股对比
同时监控同板块多只股票情绪，找出情绪龙头。

```bash
for code in 600875 601727 600089; do
    echo "=== $code ==="
    python scripts/analyze_sentiment.py --code $code --pages 1
done
```

### 定时任务
设置cron定时获取情绪数据，记录历史变化。

```bash
# 每30分钟获取一次
crontab -e
*/30 * * * 1-5 cd /path/to/guba-sentiment && python scripts/analyze_sentiment.py --code 600875 >> /tmp/sentiment.log
```

### 数据库存储
将情绪数据存入数据库，建立情绪指数历史库。
